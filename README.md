# COBRA- • Sistema de Gerenciamento de Cobrança

Sistema profissional para gestão de dívidas, cálculo automático de juros e disparos de notificações via WhatsApp.

## 🚀 Funcionalidades
- **Conexão Firebase Realtime:** Sincronização em tempo real de devedores e histórico.
- **Cálculo de Juros Diários:** Atualização dinâmica do valor da dívida baseada em porcentagem.
- **Notificações Automáticas:** Sistema de log de cobranças e integração direta com WhatsApp.
- **Gestão de Documentos:** Armazenamento em Base64 de fotos de perfil, documentos e comprovantes bancários.
- **Autenticação Segura:** Acesso controlado via repositório de credenciais centralizado.

## 📁 Estrutura de Arquivos
- `index.html`: Interface principal (UI/UX) e lógica de visualização.
- `bd.js`: Driver de conexão e operações CRUD com Firebase.
- `rec_senha.js`: Gestão de usuários, senhas e recuperação de acesso.
